LOGGING OUTPUT LOCATIONS
================================================================================

All new logging goes to:
- print() statements with file=sys.stderr for feature builders
- print() statements for inference logging (standard output)
- traceback.print_exc() for full error stacks

These will appear in:
- Docker logs: docker logs <container_id>
- Local terminal: When running "python main.py"
- Systemd journal: journalctl -u agile-impact-service

Example grep commands to view logs:
  grep "\[BUILD_" logs.txt           # Feature construction logs
  grep "\[QUALITY" logs.txt          # Quality model logs
  grep "\[EFFORT" logs.txt           # Effort model logs
  grep "\[SCHEDULE" logs.txt         # Schedule risk logs
  grep "\[PRODUCTIVITY" logs.txt     # Productivity model logs
  grep "\[ERROR\]" logs.txt          # All errors with context
