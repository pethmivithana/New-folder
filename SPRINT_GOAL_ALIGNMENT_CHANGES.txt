ANALYSIS LAYERS:
  1. Critical Blocker Detection
     - Checks for production emergency keywords
     - Only triggers if Priority = Critical/Blocker
     
  2. Semantic Similarity Analysis
     - Token overlap between sprint goal and requirement
     - Jaccard similarity scoring
     - Categories: HIGHLY_RELEVANT / TANGENTIAL / UNRELATED
     
  3. Metadata Traceability
     - Epic alignment check
     - Component tag overlap analysis
     - Structured relationship examination
     
  4. Integrated Recommendation
     - Combines all 3 layers
     - Outputs: ACCEPT / CONSIDER / EVALUATE / DEFER
     - Includes actionable next steps

RECOMMENDATIONS:
  ✓ ACCEPT — Add to sprint immediately
  ✓ CONSIDER — Discuss in team standup
  ✓ EVALUATE — Careful review required
  ✓ DEFER — Save for future sprint

RESULT DISPLAY:
  ✓ Color-coded by recommendation (green/blue/amber/red)
  ✓ Shows recommendation badge

  ✓ Provides next steps

  ✓ Manual override toggle

MANUAL OVERRIDE:
